# Trading Analysis Skill

## Workflow
1. **Identify the asset**: Determine symbol, asset class, and relevant markets
2. **Multi-timeframe analysis**: Check weekly → daily → 4H → 1H for trend alignment
3. **Key levels**: Identify support/resistance, supply/demand zones
4. **Indicators**: RSI (overbought/oversold), MACD (momentum), Volume (confirmation)
5. **Catalyst check**: Upcoming earnings, economic events, news catalysts

## Best Practices
- Always start with the higher timeframe for trend direction
- Confluence: 2+ signals agreeing = higher conviction
- Volume confirms breakouts — low volume breakouts often fail
- Risk/reward minimum 1:2 for swing trades, 1:1.5 for day trades
- Never chase — wait for pullback to key level

## Output Format
Provide: Trend (bullish/bearish/neutral), Key levels, Entry zone, Stop loss, Targets, Confidence (1-5)
