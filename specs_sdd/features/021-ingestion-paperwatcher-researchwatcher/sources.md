## Source Date Policy

- Current decision evidence: 2026 papers, current official project docs and
  current local benchmark results.
- Method/tool references: older papers such as Docling 2025, MinerU 2024,
  LightRAG 2024 and AutoRAG 2024. These remain useful, but require current
  repository review plus Matrix benchmark evidence before becoming defaults.
