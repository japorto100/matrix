# Subfeatures

## 019.1 Embedding Provider Layer

OpenRouter/OpenAI-compatible remote embeddings, deterministic test embeddings
and opt-in local HF embeddings with HDD cache.

## 019.2 Hybrid Retrieval Core

Vector/BM25 candidates, KG/path candidates and late fusion with attribution.

## 019.3 Context Bubble And Verification

Token-budgeted context assembly, diversity gates, source refs and unsupported
claim detection.

## 019.4 GraphRAG Candidate Evaluation

LightRAG, HippoRAG2, LinearRAG and E2GraphRAG are benchmark candidates, not
automatic dependencies.
