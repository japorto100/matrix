# Gates

- G001 Visual capture requires explicit consent or a documented trusted source.
- G002 OCR/layout output keeps source refs and coordinates.
- G003 Low-confidence extraction is not injected as memory.
- G004 Visual memory summaries never become KG truth without promotion.
- G005 Stale visual memories expose age and capture source.
- G006 Optical compression experiments cannot replace text/audit evidence.
- G007 Redaction runs before storage or model exposure where configured.
- G008 Meta-Harness includes visible, hidden and stale visual-memory cases.
- G009 Live extraction artifacts are inspectable and deletable.
